# Epics

## E1. Frontend Bootstrap and Buyer App Foundation
Establish the Next.js frontend codebase, design tokens, wallet integration scaffolding, routing, and baseline testing to enable rapid feature implementation.

## E2. Backend Bootstrap and Service Foundation
Establish backend service, API structure, admin auth scaffolding, contract integration adapters, and observability hooks.

## E3. Shared Tooling, CI, Quality Gates, and Environment Management
Define shared repository standards for formatting, linting, tests, CI workflows, env templates, and developer experience.

## E4. Database and Prisma Foundation
Define relational schema, migrations, Prisma client setup, seed flow, and query conventions for operational/read-model data.

## E5. Auth and Session Foundation
Implement admin authentication/session lifecycle, role-based access control, and secure token handling.

## E6. Core Product Features (Buyer + Admin + Indexing)
Implement the Delegate and DA purchase experiences, referral workflows, order indexing/read models, and admin operational modules.

## E7. Polish, Monitoring, and Non-Critical Enhancements
Complete UX polish, reliability hardening, dashboards/alerts, reconciliation tooling, and release readiness improvements.
