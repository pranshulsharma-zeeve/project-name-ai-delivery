# Frontend (Buyer App Foundation)

## Purpose
Bootstrap Next.js buyer-facing app for GPT Protocol Node Sale platform.

## Setup
```bash
npm install
npm run dev
```

## Commands
- `npm run lint` — lint checks
- `npm run test` — unit tests (Vitest)
- `npm run build` — production build validation

## Environment
Create `.env.local` from `.env.example` and set network/config placeholders.
